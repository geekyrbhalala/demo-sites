# Luxe — Salon & Spa

A premium hair salon and day spa website with a soft, organic, feminine luxury aesthetic. Rose gold meets botanical garden.

## Design

- **Color Palette**: Soft blush pink (#f4e4de), rose gold (#b76e79), deep forest green (#2d4a3e), warm white (#fefaf6), charcoal (#2c2c2c), sage (#9cac8b)
- **Typography**: Libre Baskerville (headings), Jost (body), Sacramento (decorative script)
- **Aesthetic**: Organic luxury, botanical, soft and airy

## Tech Stack

- Vue 3 (Composition API)
- Vue Router 4
- Vite (build tool)
- Custom CSS (no frameworks)
- Google Fonts

## Pages

| Route | Page | Description |
|-------|------|-------------|
| `/` | Home | Hero, services preview, before/after, team, reviews, promotions |
| `/services` | Services | Expandable service categories with pricing |
| `/booking` | Booking | Multi-step booking form with progress indicator |
| `/gallery` | Gallery | Masonry grid with category filters and lightbox |
| `/about` | About | Salon story, values, team profiles, certifications |
| `/contact` | Contact | Contact form, location, hours |
| `/login` | Login | Airy login form with social auth |
| `/register` | Register | Registration with client/staff profile selection |
| `/admin` | Admin | Appointments, client stats, revenue dashboard |

## Features

- Soft fade-up animations on scroll
- Floating botanical decorative SVG elements
- Multi-step booking form with slide transitions
- Masonry gallery with lightbox
- Rose gold hover effects and border reveals
- Full form validation with visual feedback
- Responsive design (mobile, tablet, desktop)
- Hamburger menu on mobile

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
